# Hamburgers

Un steak de bœuf maigre, cuit sans sel, ail ni oignon, peut constituer une friandise de grande valeur occasionnelle. Les problèmes surviennent avec le fromage, les condiments, les cornichons, les oignons et les pains riches en matières grasses. Certaines sauces contiennent du xylitol. Pour les animaux, le hamburger le plus sûr n'est qu'un petit morceau de viande nature, pas le sandwich assemblé.
