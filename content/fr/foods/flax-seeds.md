# Graines de lin

Les graines de lin moulues sont une source sûre de fibres et d'acides gras oméga-3 pour les chiens. Les graines de lin entières peuvent être éliminées sans être digérées, il est donc préférable de les moudre pour améliorer l'absorption.

Les chats peuvent consommer une très petite quantité de graines de lin moulues occasionnellement. Maintenir la quantité faible pour éviter les selles molles.
