# Pois gourmands

Les pois gourmands sont un légume croquant et sucré qui est sûr pour les chiens avec modération. Toute la cosse est comestible mais peut être fibreuse. Certains chiens les avalent entiers, surveillez-les pour éviter les risques d'étouffement.

Les chats peuvent manger un ou deux pois gourmands natures. Évitez les pois gourmands assaisonnés ou sautés. Comme pour toutes les friandises, limitez les portions.
