# Herbe à chat

L'herbe à chat est une plante de la famille des menthes célèbre pour son effet sur les chats. Elle est non toxique pour les chiens comme pour les chats. De nombreux chats aiment se frotter, se rouler dedans ou la grignoter, tandis que les chiens sont généralement indifférents.
