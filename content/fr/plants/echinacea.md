# Échinacée

L'échinacée, ou rudbeckie, est une plante vivace indigène robuste aux pétales violets retombants. Elle est sans danger pour les animaux et est populaire dans les produits à base de plantes pour le soutien immunitaire.

Bien que la plante elle-même soit non toxique, les compléments concentrés ne doivent être utilisés que sur indication d'un vétérinaire.
