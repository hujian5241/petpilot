# Asclépiade commune

L'asclépiade est une fleur sauvage indigène importante pour les monarques, mais elle est dangereuse pour les chiens et les chats. La plante contient des cardénolides qui affectent le cœur.

Les animaux qui mâchouillent les feuilles ou les tiges d'asclépiade peuvent tomber gravement malades. Si vous cultivez de l'asclépiade pour les pollinisateurs, placez-la dans un endroit inaccessible aux animaux.
