# Honig

Honig ist für die meisten Hunde in sehr kleinen Mengen unbedenklich, aber er enthält viel Zucker. Er sollte nicht an diabetische oder übergewichtige Tiere verfüttert werden.

Katzen können einen kleinen Geschmackstest bekommen, aber sie benötigen keine Süßigkeiten. Honig sollte nicht an Kätzchen verfüttert werden.
