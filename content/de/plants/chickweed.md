# Vogelmiere

Vogelmiere ist eine einjährige Pflanze der kühlen Jahreszeit mit kleinen weißen sternförmigen Blüten. Sie ist für Haustiere unbedenklich und wird oft von Hühnern, Kaninchen und grasenden Hunden gerne gefressen.

Wenn Sie Vogelmiere als Snack anbieten, achten Sie darauf, dass sie aus einem Gebiet stammt, das frei von Herbiziden, Pestiziden und Kot anderer Tiere ist.
